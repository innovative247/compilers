================================================================================
                        SETTING UP WSL UBUNTU
================================================================================

Before starting: Open a Windows Terminal


--------------------------------------------------------------------------------
STEP 0: Prerequisites
--------------------------------------------------------------------------------
Make sure you're running commands in a Windows Terminal (PowerShell or CMD).


--------------------------------------------------------------------------------
STEP 1: Uninstall (wipe) Ubuntu
--------------------------------------------------------------------------------
wsl --shutdown
wsl --unregister Ubuntu


--------------------------------------------------------------------------------
STEP 2: Install Ubuntu
--------------------------------------------------------------------------------
wsl --install -d Ubuntu

    Ubuntu will launch - create your username/password when prompted.


--------------------------------------------------------------------------------
STEP 3: Disable Windows PATH in WSL (optional)
--------------------------------------------------------------------------------
sudo bash -c 'cat >> /etc/wsl.conf << EOF
[interop]
appendWindowsPath = false
EOF'

    Then restart WSL:
    - Close Ubuntu session:
        exit
    - Start Ubuntu session (from Windows PowerShell):
        wsl -d Ubuntu


--------------------------------------------------------------------------------
STEP 4: Adjust prompts (optional)
--------------------------------------------------------------------------------

Normal user prompt (green username$):
    sed -i '/^export PS1=/d' ~/.bashrc
    echo "export PS1='\[\e[32m\]\u\[\e[0m\]\$ '" >> ~/.bashrc
    source ~/.bashrc

Root prompt (red root#):
    sudo sed -i '/^export PS1=/d' /root/.bashrc
    sudo bash -c "echo \"export PS1='\\[\\e[31m\\]\\u\\[\\e[0m\\]# '\" >> /root/.bashrc"

Always start in Linux home:
    echo "cd ~" >> ~/.bashrc
    source ~/.bashrc


--------------------------------------------------------------------------------
STEP 5: Install Node.js
--------------------------------------------------------------------------------
    Reopen Ubuntu first, then run:

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs


--------------------------------------------------------------------------------
STEP 6: Upgrade existing packages
--------------------------------------------------------------------------------
sudo apt update && sudo apt upgrade -y && sudo apt autoremove -y


--------------------------------------------------------------------------------
STEP 7: Install git
--------------------------------------------------------------------------------
sudo apt install -y git


--------------------------------------------------------------------------------
STEP 8: Clone git compiler repo
--------------------------------------------------------------------------------
cd ~
git clone https://github.com/innovative247/compilers.git


--------------------------------------------------------------------------------
STEP 9: Install SVN
--------------------------------------------------------------------------------
sudo apt install -y subversion


--------------------------------------------------------------------------------
STEP 10: Pull down an SVN repo
--------------------------------------------------------------------------------
mkdir -p ~/ir_local
cd ~/ir_local
svn checkout http://10.130.61.10/work2/svn/repos/SBN_IR/trunk .


--------------------------------------------------------------------------------
STEP 11: Install Claude CLI (Optional)
--------------------------------------------------------------------------------
sudo npm install -g @anthropic-ai/claude-code
